# 📖 옵시디언 사용법 (처음 시작 가이드)

> How-to-CS vault를 쓰기 위한 최소 사용법. 위에서부터 한 번만 따라하면 된다.

## 0. 개념 한 줄 정리
- **vault(볼트)** = 노트들이 담긴 폴더. 여기선 `How-to-CS` 폴더가 vault.
- **노트** = 그냥 `.md` 파일 한 개. (개념 하나 = 노트 하나)
- **[[링크]]** = 노트끼리 연결. 이게 모이면 **그래프**가 된다.

## 1. 설치 & 열기
1. https://obsidian.md 에서 다운로드 → 설치
2. 실행 → **"Open folder as vault"** 클릭
3. `C:\Users\NEXV\Desktop\Projects\How-to-CS` 폴더 선택
4. "Trust author and enable plugins?" 물으면 → **Trust** 클릭

## 2. Dataview 플러그인 설치 (대시보드 자동집계용 · 1분)
1. 좌측 아래 ⚙️(설정) 클릭
2. **Community plugins** → "Turn on community plugins" → **Browse**
3. 검색창에 `Dataview` → 설치(Install) → 활성화(Enable)
4. 설정 닫기. 이제 📊 대시보드가 숫자를 자동으로 채운다.

## 3. 둘러보기
- 좌측 **파일 목록**에서 `00_INDEX/🏠 Home` 더블클릭 → 여기가 시작점
- Home의 파란 글씨(`[[...]]`)를 클릭하면 그 노트로 이동
- 좌측 사이드바의 **● (Graph view)** 아이콘 클릭 → 점·선으로 된 지식 그래프가 보임
  - 점 = 노트, 선 = 링크. 8개 덩어리(카테고리)로 뭉쳐 있으면 정상

## 4. ⭐ 공부 체크하는 법 (매일 하는 핵심 동작)
공부한 노트를 열면 맨 위에 이런 부분이 있다:
```
status: 안함
```
이 값을 상황에 맞게 **글자만 바꾸면** 끝. (Obsidian은 자동 저장)
- 공부 시작 → `status: 공부중`
- 다 봤음 → `status: 완료`
- 까먹을 때쯤 다시 볼 것 → `status: 복습`

→ 바꾸면 📊 진도 대시보드의 진도율과 "다음에 볼 것"이 자동으로 갱신된다.

> 💡 맨 위 `---` 사이 영역(frontmatter)이 표로 예쁘게 안 보이고 코드처럼 보여도 정상.
> 그냥 `안함`을 `완료`로 고치면 된다.

## 5. 공부 순서
[[🗺️ 학습 로드맵]] 또는 [[📊 진도 대시보드]]를 열어서,
**🔴 Tier 1(빈출)** 의 `안함` 개념부터 하나씩 → `완료`로 바꿔가며 진행.

## 6. 새 개념 직접 추가 (나중에)
1. 추가할 카테고리 폴더(예: `01_운영체제`)를 우클릭 → New note
2. `_templates/개념노트.md` 내용을 복사해 붙여넣기
3. 본문에서 다른 개념을 `[[개념이름]]`으로 연결 → 그래프에 자동 편입
