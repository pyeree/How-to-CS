# 🗂 데이터베이스 지도

> `데이터베이스` 카테고리의 모든 개념. 🔴빈출 🟡중요 🟢심화

```dataview
TABLE WITHOUT ID file.link AS 개념, status AS 상태, priority AS 빈도
FROM "03_데이터베이스"
WHERE file.name != this.file.name
SORT priority ASC, file.name ASC
```
